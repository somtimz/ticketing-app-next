## Project Overview

This is the `ticketing-app-next` directory, intended for the next iteration/evolution of the ticketing application. The existing `ticketing-app` project (in the adjacent directory) is a full-stack ticketing application with role-based access control.

**Existing Project Reference:** The original ticketing app is located at `../ticketing-app/` and includes:
- Backend: Node.js/Express + SQLite (sql.js) + session-based auth
- Frontend: React + Vite + Tailwind CSS + React Router
- Features: Support ticket creation, agent assignment, comments, status workflow (Open → In Progress → Resolved)
